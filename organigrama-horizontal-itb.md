# Flujos de Trabajo Colaborativo

| De Equipo | A Equipo | Flujo de trabajo |
|-----------|----------|-----------------|
| Dirección Estratégica | Gestión Interna | Asigna objetivos administrativos y legales, define políticas de contratación |
| Dirección Estratégica | Tecnología | Define necesidades de sistemas, prioriza inversiones tecnológicas |
| Dirección Estratégica | Operaciones | Aprueba inspecciones, asigna recursos, define estándares de calidad |
| Gestión Interna | Operaciones | Provee soporte administrativo y legal, gestiona contratos con clientes |
| Tecnología | Operaciones | Sistemas y soporte para inspecciones, mantenimiento preventivo de equipos |
| Operaciones | Tecnología | Reportes técnicos, requerimientos de mantenimiento, feedback de usuarios |
| Operaciones | Dirección Estratégica | Reportes de resultados, issues de clientes, oportunidades de mejora |
| Gestión Interna | Dirección Estratégica | Reportes financieros, estado de cumplimiento, necesidades de personal |

# Alianzas Internas
- **Operaciones + Tecnología:** Sinergia para mantenimiento de equipos tecnológicos y soporte en campo
- **Dirección + Gestión Interna:** Alineación estratégica con operativa diaria
- **Gestión + Operaciones:** Soporte legal y administrativo para ejecución de inspecciones
- **Tecnología + Dirección:** Desarrollo de soluciones tecnológicas estratégicas para el negocio


# Comparativo Estructuras

| Aspecto | Vertical (Jerárquico) | Horizontal (Colaborativo) |
|---------|----------------------|---------------------------|
| Énfasis | Mando y control | Objetivos compartidos |
| Comunicación | Ascendente/descendente | Entre equipos horizontales |
| Decisiones | Por nivel jerárquico | Por consenso en equipo |
| Flexibilidad | Baja | Alta |
| Innovación | Limitada por estructura | Fomentada por colaboración |
