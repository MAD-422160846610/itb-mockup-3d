## Espaciado Tipográfico

### Escala de Espaciado

| Nombre | Equivalencia | Uso |
|--------|--------------|-----|
| xs | 4px | Entre letras muy cercanas |
| sm | 8px | Entre palabras en misma línea |
| md | 16px | Entre elementos relacionados |
| lg | 24px | Entre párrafos |
| xl | 32px | Entre secciones |
| xxl | 48px | Entre páginas |

### Ejemplo de Uso

```
Montserrat Bold 48px
(márgen-bottom: 24px)

Montserrat SemiBold 36px
(márgen-bottom: 16px)

Open Sans Regular 16px
(márgen-bottom: 16px)

Open Sans Regular 14px
(márgen-bottom: 8px)
```

## Guía de Uso

### SIEMPRE

- Usar Montserrat para TODOS los títulos
- Usar Open Sans para el cuerpo de texto
- Mantener line-height entre 1.5-1.6 para cuerpo
- Usar negrita con moderación (no todo en bold)
- Dejar espacio suficiente entre párrafos

### NUNCA

- NO usar más de 2 fuentes diferentes
- NO usar fuentes Times New Roman o Arial (genéricas)
- NO escribir todo en mayúsculas continuas
- NO usar tamaños menores a 12px
- NO exceder 75 caracteres por línea (cuerpo)
