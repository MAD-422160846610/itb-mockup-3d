## B) IDENTIDAD DE MARCA

### B1. Naming y Dominio
- [x] Evaluar nombre actual ITB
- [x] Verificar disponibilidad de dominio .com, .net, .es
- [x] Si es necesario, brainstorm de alternativas
- [x] Decidir naming final y dominio

### B2. Tipografía
- [x] Seleccionar fuente principal (headings)
- [x] Seleccionar fuente secundaria (body text)
- [x] Definir jerarquía tipográfica
- [x] Establecer tamaños y espaciados
- [x] Crear guía de uso tipográfico

### B3. Paleta de Colores
- [x] Definir color primario (brand main)
- [x] Definir color secundario (accent)
- [x] Definir color neutral (fondos, texto)
- [x] Crear paleta extendida (3-5 colores)
- [x] Establecer significados/emoción de cada color
- [x] Definir modos claro/oscuro

### B4. Logotipo e Isotipo
- [x] Diseñar logotipo principal (wordmark + symbol)
- [x] Crear isotipo (versión simplificada solo símbolo)
- [x] Diseñar marca de agua (watermark)
- [x] Definir variaciones (horizontal, vertical, símbolo solo)
- [x] Establecer zona de protección (clear space)
- [x] Definir tamaños mínimos de uso

### B5. Estrategia de Vectorización
- [x] Exportar en formatos AI, EPS, SVG
- [x] Crear variantes para diferentes fondos
- [x] Definir versiones en escala de grises
- [x] Preparar versiones monocromáticas

### B6. Sistema de Iconos
- [x] Seleccionar familia de iconos (custom vs. stock)
- [x] Definir estilo (line, filled, mixed)
- [x] Establecer tamaño y stroke estándar
- [x] Crear iconos custom para servicios clave
- [x] Desarrollar guía de uso de iconos

### B7. Mockups
- [x] Crear mockup de tarjeta de presentación
- [x] Diseñar membreted letterhead
- [x] Crear firma de email (email signature)
- [x] Diseñar brochure/flyer (1 cara)
- [x] Mockup de perfil de redes sociales
- [x] Diseñar banner para LinkedIn/Facebook
- [x] Crear presentación corporativa (template)
