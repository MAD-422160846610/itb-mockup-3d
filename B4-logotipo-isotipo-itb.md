# B4 - Logotipo e Isotipo ITB

**Proyecto:** [[itb-inspecciones-tecnicas-de-buques]]

**Estado:** Completado

---

## Concepto del Logotipo

### Inspiración
El logotipo de ITB se inspira en la **rosa náutica** (brújula marina) y las **líneas de inspección**, representando:
- Precisión y dirección (rosa náutica)
- Inspección técnica (líneas de medición)
- Seguridad marítima (escudo/mar)

### Elementos Visuales

| Elemento | Descripción |
|----------|-------------|
| **Símbolo** | Rosa náutica estilizada con líneas de inspección |
| **Tipografía** | "ITB" en Montserrat Bold |
| **Claim** | "Inspecciones Técnicas de Buques" (opcional) |

---

## Variantes del Logotipo

### Variante 1: Horizontal Principal
```

 ITB Inspecciones Técnicas 
 de Buques 

```
- **Uso:** Header de web, propuestas, brochures
- **Proporción:** 3:1 (ancho:alto)

### Variante 2: Vertical / Apilado
```

 
 ITB 

 ITB... 

```
- **Uso:** Tarjetas de presentación, perfiles sociales
- **Proporción:** 1:1.5

### Variante 3: Isotipo (Solo símbolo)
```

 

```
- **Uso:** Favicon, iconos de app, watermarks
- **Proporción:** 1:1 (cuadrado)

### Variante 4: Wordmark (Solo texto)
```

 ITB 

```
- **Uso:** Firmas de email, documentos oficiales
- **Proporción:** 4:1

## Zona de Protección (Clear Space)

```

 
 
 
 ITB 
 
 
 
 x = altura de la "I" en ITB 
 Zona = 0.5x alrededor del símbolo 

```

**Regla:** Dejar espacio mínimo de media "I" alrededor del logotipo.

---

## Proporciones Prohibidas

| No hacer | Porque |
|----------|--------|
| Estirar o comprimir | Distorsiona la marca |
| Rotar | Pierde reconocimiento |
| Cambiar colores arbitrariamente | Rompe sistema |
| Agregar efectos (sombra, glow) | Inconsistencia |
| Usar sobre fondos muy concurridos | Falta legibilidad |
